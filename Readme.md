# 实习项目：三维模型显示系统

使用Qt+DirectX12编写

本项目地址：https://github.com/Valhally/ModelViewer

如需编译此工程, 你应：

1. 安装Visual Studio 2019
2. 安装Qt5或更高版本的源码并在安装时选择相应的qt for msvc 64bit组件
3. 在VS2019的扩展市场中安装Qt Visual Studio Tools
4. 进入Qt拓展设置页面，设置Qt源码的安装路径
5. 重启Visual Studio，进入该ModelViewer项目的属性页面，在Qt Project Settings选项卡中设置项目使用的Qt版本号
6. 一切结束，现在应该能够编译该工程
